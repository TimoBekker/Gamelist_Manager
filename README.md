# Gamelist_Manager

Верхняя панель:

Платформа — выпадающий список для выбора платформы (например, Steam, NES, PS1 и т.д.).
Поиск — поле для поиска игр по названию.
Категория — фильтр по подпапкам/жанрам внутри платформы.
Файлы — фильтр: показывать все игры, только с файлами, только без файлов.
Починить названия — исправляет названия игр (убирает нумерацию, приводит артикли к началу).

Основная область:

Таблица — список всех игр выбранной платформы с чекбоксами, названием и жанром.
Можно выделять одну или несколько игр.
Двойной клик — отметить/снять чекбокс.
Правая кнопка мыши — открыть папку с файлом игры.
Изображение — обложка выбранной игры (или заглушка, если нет картинки).
Описание игры — текстовое поле с описанием, жанром, разработчиком, издателем, датой выхода и числом игроков.

Блок управления:

Перемещать файлы в папку — чекбокс: если включён, удаляемые файлы будут перемещаться в папку "Удалённые", а не удаляться навсегда.
Удалить выбранную игру — удаляет одну выделенную игру и все связанные файлы.
Удалить отмеченные (N) — удаляет все отмеченные (чекбоксами) игры и их файлы.
Удалить лишнее — удаляет из gamelist.xml записи игр без файлов.
Почистить media — удаляет с диска все неиспользуемые файлы/папки, не упомянутые в gamelist.xml.
Загрузить из Steam — загружает описание и обложку из Steam для выбранных или всех игр.
